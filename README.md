### 3. Sesuaikan Warna
Edit variabel CSS di `style.css`:
```css
:root {
    --primary-color: #FF6B35;      /* Warna utama */
    --secondary-color: #004E89;    /* Warna sekunder */
    --accent-color: #F7931E;       /* Warna accent */
    /* dll */
}
```

### 4. Upload ke Server
- Buat akun di hosting (Niagahoster, Hostinger, DLL)
- Upload ketiga file (index.html, style.css, script.js) ke server
- Akses melalui domain Anda

## 🔧 Customization Guide

### Mengubah Text & Content
1. Buka `index.html` dengan text editor (VS Code, Notepad++, dll)
2. Temukan text yang ingin diubah
3. Edit dan simpan (Ctrl+S)
4. Refresh browser untuk melihat perubahan

### Mengubah Warna
1. Buka `style.css`
2. Cari variabel di `:root` (bagian atas file)
3. Ubah nilai hex color
4. Contoh: `--primary-color: #FF6B35;` → `--primary-color: #FF0000;`

### Menambah Section Baru
1. Copy struktur section yang sudah ada
2. Paste sebelum `</body>`
3. Ubah ID dan class sesuai kebutuhan
4. Tambahkan styling di `style.css` jika diperlukan

### Menambah Font Custom
1. Edit bagian `<head>` di `index.html`
2. Import font dari Google Fonts atau source lain
3. Update font-family di `style.css`

## 📊 Analytics & SEO

Untuk hasil maksimal, tambahkan:

### Google Analytics
```html

<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
```

### Google Search Console
1. Daftar di Google Search Console
2. Verify domain Anda
3. Submit sitemap (jika ada)

Sudah included di `index.html`:
- `<meta name="description">` - Deskripsi untuk Google
